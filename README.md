# Tugas Algoritma 2

Repository ini digunakan sebagai wadah untuk mengumpulkan dan menyimpan seluruh tugas mata kuliah Algoritma 2.

Isi repository berupa implementasi program, latihan, dan eksplorasi berbagai konsep algoritma yang dikerjakan selama proses perkuliahan.

## Tujuan

* Mendokumentasikan hasil tugas Algoritma 2
* Melatih pemahaman konsep algoritma melalui praktik langsung
* Menyimpan kode sebagai referensi pembelajaran

## Catatan

Repository ini akan terus diperbarui seiring dengan penambahan tugas dan materi selama perkuliahan berlangsung.
